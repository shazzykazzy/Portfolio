# Quick Start Guide

Get your Tutoring Invoice Manager up and running in 5 minutes!

## Installation (3 minutes)

1. **Open a terminal in the tutoring-invoicing folder**

2. **Install Python dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Start the application:**
   ```bash
   python app.py
   ```

4. **Open your browser:**
   - Go to: http://localhost:5000

That's it! The app is running.

## Daily Workflow

1. **After each session:** Log it in the Sessions page
2. **End of week/month:** Create invoices from unbilled sessions
3. **When paid:** Mark invoices as paid
4. **Monthly:** Check Reports for income summary
